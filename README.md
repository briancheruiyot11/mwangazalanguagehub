# Mwangaza Language Hub 🌍
Welcome to **Mwangaza Language Hub**, a modern and user-friendly **Learning Management System (LMS)** designed to help learners discover and master Kenyan languages. Built as a full-stack project by a passionate developer.
Mwangaza simplifies language learning by connecting learners with structured courses in Kenyan languages like Kikuyu, Luo, Kalenjin, Luhya, Kamba and more  all in one platform.

## 🌐 Live Demo
Deployed on **Vercel** – [Visit Mwangaza Language Hub](https://mwangazalanguagehub.vercel.app/)

## 💠 Technologies Used
- **Next.js** - React framework for routing & SSR
- **Tailwind CSS** - For rapid, responsive styling
- **Flask** - Python backend framework for REST API
- **SQLAlchemy** - ORM for database management
- **PostgreSQL** - Production database
- **Flask-JWT-Extended** - Authentication with JSON Web Tokens
- **Render** - Deployment for the backend - https://mwangazalanguagehub.onrender.com
- **Vercel** - Deployment for the frontend

## ✨ Features
* 🏠 **Landing Page**: Welcoming intro with platform overview and call-to-action
* 🔐 **Authentication**: Secure register and login with role-based access (Admin & Learner)
* 📚 **Courses Page**: Browse all available Kenyan language courses
* 🎓 **Learner Dashboard**: Track completed lessons and overall learning progress
* 📝 **Admin Dashboard**: Monitor platform stats — courses, lessons, users, and activity logs
* ➕ **Course Management**: Admins can create and manage language courses
* 📖 **Lesson Management**: Admins can add text or video lessons to courses
* ✅ **Progress Tracking**: Learners can mark lessons as complete and track their journey
* 🗂️ **Activity Logs**: Admins can view all platform activity in real time

## 📖 How to Use the LMS
**Open the app** – Visit the live app in your browser via [https://mwangazalanguagehub.vercel.app/](https://mwangazalanguagehub.vercel.app/)

### As a Learner
1. **Register** for a new account on the Sign Up page
2. **Login** with your credentials
3. **Browse** available language courses on the Courses page
4. **Click a course** to view its lessons
5. **Mark lessons as complete** to track your progress
6. **View your dashboard** for a summary of your learning journey

### As an Admin
1. **Login** with admin credentials -> The admin secret password is **Brian124**
2. **View the Dashboard** for a platform overview — users, courses, lessons, and logs
3. **Create Courses** under the Courses section
4. **Add Lessons** to existing courses via the Add Lesson page
5. **Monitor Activity** through the Activity Logs page
